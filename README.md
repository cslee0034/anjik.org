# Anjik.org

작구 제품 유해성분 확인 어플리케이션

사용한 자료는 한국소비자원과 서울시전자상거래센터에서 제공하는 공식 발표를 기반으로 하며 이메일과 전화를 통해 자료 사용의 허가를 받았습니다.

이 프로젝트는 프로그래밍을 통해 현실의 문제를 해결하고 사회에 공헌하는 것을 목표로 하고 있으며 비영리로 운영 중입니다.
